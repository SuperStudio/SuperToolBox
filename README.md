# 关于

SuperToolBox 是一款**Window 工具箱**，支持以下功能：

- URL 编码/解码
- 加解密
- 鼠标控制
- 显示设备信息
- Http Header 格式化
- 键盘控制
- 时间戳
- 进制转换

具有的特殊功能：

- 更换皮肤
- 插件管理
